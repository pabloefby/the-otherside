TheOtherside es un sitio web diseñado para que usuarios de diferentes partes de la República Mexicana compartan anécdotas, historias, leyendas y experiencias relacionadas con temas paranormales. Entre las categorías principales se encuentran: leyendas, avistamientos de extraterrestres, fantasmas, conspiraciones, entre otras.

El proyecto está desarrollado con las siguientes tecnologías:
	•	Frontend: React, ubicado en el directorio front/.
	•	Backend: Servidor construido con ExpressJS en la carpeta server/.
	•	Base de datos: MySQL, con toda la información necesaria para el funcionamiento del sitio, ubicada en el directorio mysql/.
